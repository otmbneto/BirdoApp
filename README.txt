##BirdoAPP##
leobaza@ birdo 2021

--------------
1 - main App na raiz 
2 - app => utils do python e config scripts para criar paths do app
3 - config => arquivos de config dos projetos e o menu.xml das versoes do toon boom
4 - extra => aplicativos usados pelo birdoAPP (ffmpeg..)
5 - gui => ui do mainApp
6 - package => birdoPack do toon boom (baixar do servidor ultima versao e deixar aqui)
7 - templates => toon boom templates para reuso
8 - venv => virtual enviremento do python
9 - app.json => info das versoes do app e do package
10 - BirdoApp.py => main app script
11 - birdoLocal => info dos projetos (talvez mude pra dentro de config)
12 - requiriments.txt => com modulos a serem instalados no virtual envirement do python
