Neste folder ficam os scripts que iniciam o menu da birdo (depende do menus.xml no folder de instalacao do toon boom!);

siga as 3 regras:
 1 - nome de arquivo js no padrao 1-Nome_Script.js (Com _ separando as palavras, pois o nome vira o nome do item no menu trocando os '_' por espaco);
 2 - .tooltip com o mesmo nome do js
 3 - Nome da funcao dentro do .js tem q ser o mesmo nome do item sem espaco ou _ (NomeScript);
